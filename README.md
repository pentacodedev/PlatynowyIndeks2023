## Wymagania
  Front end wymaga frameworku .net 7.0, a back-end - node.js (projekt testowany na wersji v19.6.1)

## Uruchamianie
aby uruchomić projekt należy wykonać następujące polecenia:


### front-end
  1. `npm install`
  2. `ng serve`

### back-end
  1. `dotnet run`
